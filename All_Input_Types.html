HTML Input Types Demo

Input type - text:
A single-line text field. The value attribute defines the
value of the input field.
    
Input type - password:
An input field can be used to enter a password.

Required:
The required attribute specifies that user input is a must.
If the user does not enter any value in the input field which
is associated with this attribute, a default error message appears
on the screen.
    
Input type - email:
• An input field that accepts email addresses.
• It has in-built validation for an email.
    
Input type - number:
• Defines an input text box for only numerical input.
• Gives an error if values go beyond min/max limits or include invalid characters.
• Attributes: min, max, and step.
    
Input type - checkbox:
• Defines checkboxes.
• Multiple options can be selected.
    
Input type - radio:
• Defines radio buttons.
• Only one option can be selected per group (same name).
    
Input type - file:
Creates a control to upload files to the server.
    
Input type - color:
Defines a color picker.
    
Input type - date:
Creates a calendar picker to select a date.
    
Input type - week:
Defines a week picker.
    
Input type - month:
Defines a month picker.
